# Staff Leave Management System

A Laravel 10 based internal web application for managing staff leave requests with role-based access for Employees, Managers, and Admins.

This system helps employees apply for leave, managers approve/reject requests, and admins manage users, leave types, and organization-wide leave records.

---

# Features

## Employee

- Register and Login
- Apply for Leave
- View Own Leave Applications
- Cancel Pending Leave Requests
- Filter Leave Requests by Status and Leave Type
- Dashboard Summary of Leave Usage

---

## Manager

- View Leave Applications of Assigned Employees
- Approve Leave Requests
- Reject Leave Requests
- Add Comments while Approving/Rejecting
- View Leave Approval History

---

## Admin

- Full CRUD for Employees and Managers
- Assign Employees to Managers
- View All Leave Applications
- Override Leave Status
- Manage Leave Types
- Configure Leave Entitlements
- Filter Applications by Status, Department, and Date Range

---

# Technology Stack

- Laravel 10
- PHP 8+
- MySQL
- Blade Templates
- Bootstrap Admin Dashboard UI
- Vanilla JavaScript
